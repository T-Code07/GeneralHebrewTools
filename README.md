# Purpose/Mission
The purpose of this project is to experiment with crossing my Hebrew knowledge and my CS knowledge. I want to build up comfortability with working with language to futher my skills in Natural Language Processing, possibly for research later on. 

## Tools and Implementations so far:
1. Prints the Unicode Hebrew alphebet starting in with Aleph
2. Finds the verb stem on very simple Pa'al verbs. (Only present tense, no weak root)
